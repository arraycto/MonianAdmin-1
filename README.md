&emsp;&emsp;正在学习写代码，想做一个后台管理系统，当作以后项目开发的基础。本系统参考了很多优秀的开源项目，系统会持续更新一些实用的功能。但是事情比较多，只能利用闲暇休息时间来进行开发，所以更新会不定时，但是我应该短期内不会弃更(也说不定，哈哈哈)。
